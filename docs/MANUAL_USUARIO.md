# 📚 Manual de Usuario - Biblioteca IA (ChacaBot)

## 1. Introducción
Bienvenido a **Biblioteca IA**, el sistema inteligente de gestión de préstamos del Colegio Chacabuco. Este sistema combina una base de datos de libros con **ChacaBot**, un asistente virtual que te ayuda a encontrar materiales rápidamente usando lenguaje natural.

## 2. Acceso al Sistema
Para utilizar el sistema, necesitas credenciales de acceso.

1.  **Ingresar al sitio web:** Abre tu navegador y ve a la dirección de la aplicación.
2.  **Pantalla de Login:** Se te pedirá Usuario y Contraseña.
    *   **Usuarios (Alumnos):** Ingresa con tu cuenta personal.
    *   **Administrador (Bibliotecario):**
        *   **Usuario:** `biblio`
        *   **Contraseña:** `biblio123`

## 3. Funcionalidades para Alumnos
Una vez dentro, tendrás acceso a la interfaz de chat.

### 💬 Chat con ChacaBot
Escribe en la barra inferior para consultar por libros.

*   **Buscar por categoría:**
    *   *Ejemplo:* "Dame libros de biología" o "¿Qué tenés de matemáticas?"
*   **Buscar por título o autor:**
    *   *Ejemplo:* "Busco el libro Don Quijote" o "Libros de Rowling".
*   **Consultar el catálogo:**
    *   *Ejemplo:* "¿Qué tipos de libros hay?" o "Dame el catálogo completo".
*   **Saludos:**
    *   Puedes saludar al bot ("Hola", "Buenas") y te responderá amablemente.

### ⚠️ Restricciones del Asistente
ChacaBot es un asistente **especializado en la biblioteca**.
*   **Lo que NO hace:** No responde sobre recetas de cocina, clima, noticias generales ni tareas escolares fuera de libros.
*   **Respuesta estándar ante otros temas:** Si preguntas "¿Cómo hago una pizza?", el bot te responderá: *"Soy ChacaBot y solo puedo ayudarte con información sobre la biblioteca..."*

## 4. Funcionalidades para Bibliotecarios
Al ingresar con el usuario `biblio`, tendrás acceso al panel de administración:

1.  **Gestión de Libros:** Visualizar la lista completa de libros, sus estados (Disponible/Prestado) y ubicación.
2.  **Dashboard:** Ver métricas básicas de uso.

## 5. Soporte
Si tienes problemas para iniciar sesión o la IA no responde correctamente, contacta a la administración de la biblioteca.
